## Features

- Detail info on a movie

## Tools & Tech
- MVVM Design Pattern
- JetPack Component
- ViewModel & Flow
- Hilt Dependency Injection
- Retrofit
- Unit Testing
- Bongo API

## Deployment
Place your bongo `API_KEY` into `gradle.properties` file and build app

```
  API_KEY="XXXXXXXXXXXXXXXXXXXXXXXX"
```
